# 재미 루프 & 페이싱 설계 — 바이블 v2 보충 (유물 6빌드 확정판)

> 본 문서는 프로젝트_바이블_v2.md의 유물 섹션을 대체하고 페이싱 기준을 추가한다.
> 충돌 시 본 문서가 우선. 모든 수치는 임시값이며 balance.ts로.

---

## 1. 핵심 루프와 박자

```
스테이지 클리어 → 더 강한 적(벽) → 방치 파밍(골드·장비·피)
→ 제단 소환·능력치 구매 → [쾌감 피크: 원하던 유물 / 5줄 전설]
→ 강해짐 체감 → 벽 돌파 → 반복
```

루프는 4개의 박자 층으로 동시에 굴러간다. 각 층마다 반드시 눈에 보이는
피드백이 하나씩 있어야 한다:

| 층 | 주기 | 박자 | 피드백 |
|---|---|---|---|
| 마이크로 | 초 | 처치 → 골드·피 튐 | 코인 포물선, 피 게이지 픽셀 상승 |
| 쇼트 | 분 | 능력치 1회 구매, 장비 1개 교체 | 데미지 숫자 자릿수 변화 |
| 미드 | 10분~ | 제단 소환 1회, 스테이지 1개 돌파 | 소환 연출, 도전 성공 팡파레 |
| 롱 | 일 | 빌드 완성(★), 챕터 보스 격파 | 변신급 연출 + 스토리 한 줄 |

## 2. 벽(Wall) 설계 — "막힘"을 파밍 지시서로

- 벽의 위치는 의도적으로 고정: **각 챕터 x-5(중간 벽), x-10(보스 벽).**
  1~4, 6~9는 술술 풀리게 — 진행감과 벽의 대비가 리듬을 만든다.
- 도전 실패는 좌절이 아니라 지시여야 한다. 실패 시 판정 리포트 1줄:
  - 시간 초과 → **"화력이 부족하다"** (→ 능력치·무기·유물 화살표)
  - 사망 → **"버티지 못했다"** (→ 방어구·HP 화살표)
- 실패 팝업에 [추천 사냥터 이동] 버튼 — 부족 계열의 드랍 편향 사냥터로
  원탭 이동. "막힘 → 뭘 할지 안다 → 방치 시작"이 10초 안에 이어진다.

## 3. 쾌감 피크 ① — 5줄 전설 (장비 갓롤)

- 등급별 줄 수: 기본스탯 1줄 + 옵션 0/1/2/3/4.
- **전설은 6번째 줄로 '죄의 옵션' 1줄 확정 부착** (영웅은 30% 확률)
  → 전설 = 항상 5줄+@, "옵션 5줄"의 쾌감이 시스템으로 보장된다.
- 죄의 옵션 = 빌드 전용 옵션 (4. 표 참조). 갓롤의 정의가 명확해진다:
  "내 빌드의 죄의 옵션 + 빌드와 시너지인 일반 옵션 4줄."
- 드랍 연출 차등: 보라 = 토스트 강조 / 주황 = 화면 1프레임 플래시 +
  0.5초 정지 + 긴 팡파레 / **장착 중인 유물과 같은 죄의 옵션이면
  팡파레에 한 음 추가** (갓롤 후보임을 귀로 먼저 안다).
- 전설은 자동판매 대상에서 제외(실수 방지 + 쾌감 보존). 큐브 합성은
  잡템 처리의 도박 쾌감으로 유지: 동일 등급 3 → 상위 1.
- 보스 반복전 정체성: 피 대량 + **해당 챕터 죄의 옵션이 붙은 장비
  드랍률 상승** — "오만 빌드 장비는 루시안을 돈다"는 D2식 보스 파밍.

## 4. 쾌감 피크 ② — 제단과 유물 (죄의 유물 6빌드 확정)

### 소환 규칙
- 처치 시 피 획득 (일반 1 / 정예 5 / 보스 100, 스테이지 계수 적용).
- 게이지 충전 → 소환 1회. 요구량은 회차마다 점증(곡선은 balance.ts).
- **미보유 유물 가중치 2배** (중복 지옥 방지).
- **소환 5회마다 '지정 소환' 1회 적립: 죄 계열을 선택해 소환.**
  "원하던 유물"이 순수 운이 아니라 도달 가능한 목표가 된다.
- 중복 = 해당 유물 ★+1 (상한 5★). **3★ 이상 승급은 해당 죄의 보스
  처치 후 개방** — "죄의 주인을 꺾어야 그 죄를 온전히 다룬다."
  (유물 풀 자체는 처음부터 6종 전부 개방 — 초반부터 빌드 선택의 재미)

### 6빌드 표 (유물 슬롯 1 — 장착 유물 = 공격 스타일 + 규칙 + 비주얼)

| 죄 | 유물/빌드 | 스타일·화면의 그림 | 기본 규칙 | 3★ 해금 | 5★ 캡스톤 | 죄의 옵션 풀 |
|---|---|---|---|---|---|---|
| 오만① | 망령 군주 | 부채꼴 휘두르기, 점눈 유령 군단 동행 | 처치한 적이 망령으로(상한 4) | 망령 사망 시 폭발 | 상한 +2, 정예도 망령화 | 망령 피해%, 망령 상한 +1 |
| 탐식② | 피의 광전사 | 초고속 연타, 붉은 픽셀이 빨려옴 | 흡혈 8%, 초당 HP 1% 누수, 타 회복 무효 | 누수 중 공속 +25% | 처치 시 누수 3초 정지 | 흡혈 +%, 누수 감소% |
| 비탄③ | 역병 의사 | 직격 0, 초록 스택·구름이 번짐 | 공격=역병 부여, 인접 전염, 사망 시 구름 | 역병 사망 = 폭발 전염 | 구름 위에서 자신 회복 | 전염 범위 +1, 역병 피해% |
| 광신④ | 순교자 | 공격마다 자기 점멸, 거대 숫자 | 공격 시 HP 2% 소모, 잃은 HP 1%당 피해 +1.5% | HP 30%↓ 치명 +25%p | 자해로는 죽지 않음(1 고정) | HP 소모 −%, 저체력 피해% |
| 심연⑤ | 처형자 | 느린 광선, 표식 적 처형 플래시 | 표식 누적, 임계 도달 시 처형(비율 피해) | 처형 시 표식 주변 전파 | 보스에도 처형 적용(축소율) | 임계 −%, 처형 피해% |
| 절망⑥ | 왕의 그림자 | 게이지 충전 → **팔레트 반전 폭주** | 폭주 N초간 광역 난무, 종료 후 탈진 | 폭주 중 처치 시 +0.5초 | 탈진 제거 | 폭주 지속 +초, 게이지 충전% |

- 구현 비용 점검: 전부 점·사각형·기존 스프라이트 재활용·팔레트 스왑으로
  표현 가능. 신규 도트는 망령(8×8 점눈 유령) 1종뿐.
- 데미지 공식과의 관계: 각 규칙은 파이프라인의 명시된 단계에 후킹
  (styleMult, 합연산 그룹, 처형은 별도 비율 피해 채널).

## 5. 복귀 모멘트 (방치의 클라이맥스)

복귀 팝업은 정산서가 아니라 루프의 한 박자다. 순서대로 연출:
1. 경과 시간 + 골드 카운트업 + 장비 요약("전리품 12개 — 보라 1!")
2. 피 게이지가 차오르는 픽셀 연출 → **가득 찼다면 "제단이 깨어났다"
   문구와 함께 제단 탭에 붉은 점등**
3. 복귀 30초 안에 소환 1회 + 능력치 몇 단계 구매가 가능하도록 피 캡과
   골드 수급을 조정 — "돌아오자마자 할 일이 기다림"이 재접속 동기.
- 방치 캡 10시간. 피는 50% 효율, 단 **캡은 정확히 소환 1회분까지만**
  (오프라인만으로 제단이 2회 이상 돌면 온라인 파밍 가치가 죽는다).

## 6. 페이싱 목표표 (balance.ts 튜닝의 북극성)

| 이정표 | 목표 시점 |
|---|---|
| 첫 유물 (첫 빌드 탄생) | 시작 5~8분 — 온보딩의 클라이맥스 |
| 첫 보라 장비 | ~15분 |
| 챕터1 중간 벽(1-5) | ~20분, 첫 "막힘→파밍→돌파" 학습 |
| 루시안(첫 보스) | 40~60분 |
| 첫 전설(5줄) | 챕터 2 구간 |
| 지정 소환 1회차 | 첫날 안에 |
| 제단 소환 주기(안정기) | 15~30분/회 (방치 포함) |
| 챕터 클리어 간격 | 1챕터 당일 → 6챕터 1~2주 (일일 방치 전제) |
| 첫 5★ 빌드 완성 | 엔드게임 ~ 레오니드 직전 |

- 리듬 원칙: **쇼트 박자(분)는 항상 무언가 살 수 있게, 미드 박자(십분)는
  항상 무언가 기다려지게.** 골드가 남아돌거나 제단이 너무 자주 열리면
  박자가 죽는다 — 둘 다 "살짝 모자라게"가 정답.

## 7. 바이블 v2 반영 사항 (요약)
- 유물 12종 표 → 본 문서 6빌드 표로 교체.
- 장비: 전설 죄의 옵션 확정 1줄, 영웅 30%. 옵션 풀에 죄의 옵션 6계열 추가.
- 제단: 미보유 가중 2배, 소환 5회당 지정 소환, 3★+ 승급은 보스 처치 조건.
- 도전 실패 리포트 2종 + 추천 사냥터 버튼.
- 복귀 팝업 연출 시퀀스, 피 오프라인 캡 = 소환 1회분.
